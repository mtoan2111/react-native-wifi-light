# react-native-wifi-light

Wifi iteractive for react-native

## Installation

```sh
npm install react-native-wifi-light
```

## Usage

```js
import WifiLight from "react-native-wifi-light";

// ...

const result = await WifiLight.multiply(3, 7);
```
